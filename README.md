# ROS 2 Camera Framework

A modular camera analysis pipeline built with ROS 2, C++, Python, and OpenCV.

This project uses camera and autofocus analysis as a practical path for learning
ROS 2 software architecture. The first release will capture images, calculate
focus and image-quality metrics, and publish results for visualization.

## Target architecture

```text
USB camera / video file
          |
          v
     camera_node  -- /camera/image_raw -->  focus_node
                                               |
                         /focus/score <--------+
                         /focus/state <--------+
                                               |
                                               v
                                      visualization / logging
```

See [docs/architecture.md](docs/architecture.md) for node responsibilities and
planned ROS interfaces.

`build/`, `install/`, and `log/` will be generated by `colcon` and are excluded
from version control.

## Development environment

- Development client: macOS
- ROS host: Ubuntu 24.04 LTS
- ROS distribution: ROS 2 Jazzy Jalisco
- Build system: `colcon`, CMake, and `ament`
- Languages: C++ and Python
- Vision stack: OpenCV and `cv_bridge`

Build and run instructions will be added after the first ROS package is created
and verified on Ubuntu.
